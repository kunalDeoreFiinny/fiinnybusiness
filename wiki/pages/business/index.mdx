# Fiinny for Business — Overview

KaranArjun SaaS (or **Fiinny for Business**) is a comprehensive retailer management platform built for Indian wholesale and distribution businesses. It handles everything from POS billing and GST invoicing to inventory, payment links, and AI-powered business insights.

import { Callout, Cards, Card } from 'nextra/components'

<Callout type="info">
**Product hierarchy:**
- **Fiinny** — Parent brand
- **Fiinny Personal** — Consumer expense tracking app (Flutter/Android)
- **Fiinny for Business (KaranArjun)** — B2B SaaS for retailers & distributors (React/Web)
</Callout>

## What It Does

| Feature | Description |
|---|---|
| POS Billing | Quick cash/UPI billing at the counter with receipt printing |
| B2B Invoicing | Full GST tax invoice generation with GSTIN validation |
| Worklist | Saved invoices pending dispatch, approval, or payment |
| Inventory & Batches | Stock management with batch tracking and barcode labels |
| Retailers CRM | Manage retail buyers with credit limits and outstanding dues |
| GST Reports | GSTR-1 / GSTR-3B style tax computation and export |
| Financial Reports | P&L, outstanding receivables, cash flow by date range |
| Payment Links | Razorpay-powered UPI/card payment links with QR codes |
| AI Advisor | GPT-4o-mini business intelligence Q&A with live Firestore data |
| Invoice Templates | Drag-and-drop invoice field customization + PDF branding |
| Multi-company | Switch between multiple business entities from one login |

## Tech Stack

```
Frontend:   React 18 + TypeScript + Vite
Styling:    Vanilla CSS (custom design system, CSS variables)
Routing:    React Router v6 (code-split lazy imports)
Backend:    Firebase Auth + Firestore + Cloud Storage
Functions:  Firebase Cloud Functions Gen 2 (Node 20)
AI:         OpenAI GPT-4o-mini via Cloud Function
Payments:   Razorpay (payment links + webhook)
PDF:        jsPDF + jspdf-autotable
i18n:       react-i18next (English + Hindi)
Monitoring: Sentry (error tracking) + UptimeRobot (uptime)
CI/CD:      GitHub Actions → Firebase Hosting
```
