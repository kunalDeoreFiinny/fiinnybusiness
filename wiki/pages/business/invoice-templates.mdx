# Invoice Templates

The Invoice Template Builder lets businesses customize how their PDF invoices look — they can add/remove fields, change the logo, and configure branding.

**Files:** `src/pages/InvoiceTemplateBuilderPage.tsx` (30KB), `src/pages/InvoiceSettingsPage.tsx` (14KB), `src/services/invoiceTemplateService.ts`, `src/utils/invoiceEngine.ts`

## What You Can Customize

- Business name, logo, address on invoice header
- GSTIN, license numbers on invoice
- Which fields to show/hide (buyer phone, email, PAN, etc.)
- Field order (drag to reorder)
- Signature line and footer text
- Template name (e.g., "Standard GST Invoice", "Cold Storage Invoice")
- Color scheme (primary color for header)
- Paper size (A4 or A5)

## Template Structure

```typescript
// From types/invoiceTemplate.ts
interface InvoiceTemplate {
  id: string;
  name: string;
  fields: InvoiceField[];
  isDefault: boolean;
  createdAt: Timestamp;
}

interface InvoiceField {
  id: string;
  label: string;       // "Buyer Phone"
  sourceKey: string;   // "retailer.phone"
  show: boolean;       // Toggle visibility
  order: number;       // Drag-drop position
  isCurrency: boolean; // Format as ₹X,XX,XXX
}

interface InvoiceTemplateBranding {
  businessName: string;
  address: string;
  gstin: string;
  licenseNumbers: string;
  logoUrl: string;
  signatureUrl: string;
  footerText: string;
  primaryColor: string;
}
```

## Multiple Templates

A business can have multiple templates (e.g., one for regular goods, one for cold-chain products with different fields). Templates are stored and can be switched per invoice:

```
tenants/{tenantId}/invoiceTemplates/{templateId}
```

## Template Gallery

`InvoiceTemplateBuilderPage.tsx` shows a **gallery of pre-built templates** to start from:

- Classic (black & white, minimal)
- Modern (colored header, logo prominent)
- Retailer (includes delivery address, vehicle number)
- Export (includes shipping details, country fields)

Once selected, the user can customize further in the drag-and-drop field editor.

## Logo Upload

Business logos are uploaded to **Firebase Storage**:
```
tenants/{tenantId}/logo.png
```

The logo URL is stored in `tenantData.logoUrl` and injected into PDFs via `doc.addImage()`.
