# Financial Reports

The Financial Reports module provides **business-level P&L, receivables, and cash flow reports** using Firestore invoice and POS data.

**File:** `src/pages/FinancialReportsPage.tsx` (21KB)

### 3. Collection Report
Payments received in a date range:

```typescript
const payments = invoices
  .filter(inv => inv.status === 'paid')
  .filter(inv => inv.paidAt >= startDate && inv.paidAt <= endDate)
  .map(inv => ({ retailer: inv.buyerName, amount: inv.grandTotal, mode: inv.paymentMode }));
```

### 4. Expense Summary
For businesses that log purchase orders, expenses are tracked and shown against revenue.

## Date Range Filters

All reports support flexible date filtering:
- **Today** / **This Week** / **This Month** / **This Quarter** / **This Year**
- **Custom Range** — date pickers for precise analysis

## Export

All financial reports can be:
- Exported as **CSV** (for Excel)
- Printed as **PDF** (with business header and date range)

## Dashboard Integration

Aggregated financial data is shown on the main Dashboard (`/dashboard`):
- Monthly revenue card with sparkline
- Outstanding receivables total
- Today's collection

The dashboard re-computes these numbers via Firestore queries on each page load (no pre-aggregated cache for simplicity).
